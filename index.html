Job Application Tracker

A personal tool I built to track my own job applications during my job search — turned into a full Kanban dashboard.

Why I built it

I was managing 30+ applications in a spreadsheet and kept losing track of follow-ups. I wanted to practise React and Firebase, so I built something I'd actually use.

What it does

Kanban board to move applications between stages. Deadline reminders, status tracking, and a simple analytics view.

What I learned

Firebase real-time sync, React Context for state management, and how to handle debounced inputs to avoid unnecessary re-renders.

Tech stack
  • React with Context API — my first real state management implementation
  • Firebase Auth + Firestore — learned auth flows and real-time data listeners
  • Tailwind CSS — first time using a utility-first CSS framework
  • Deployed on GitHub Pages
